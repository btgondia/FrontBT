# FrontBT

A front-end project using Vite and React.

## Getting Started

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm start
```

### Build for production

```bash
npm run build
```

### Run tests

Tests are run using [vitest](https://vitest.dev/). There are currently no test files included.

```bash
npm test
```
